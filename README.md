# Ditt nav

Kildekoden til frontend-appen for ny ditt nav.

# Komme i gang

For å kjøre opp app-en i dev:

1.  `npm install`
2.  `npm run build`
2.  `npm run server`

For å kjøre tester:

1.  `npm install`
2.  `npm run test`

# Henvendelser
